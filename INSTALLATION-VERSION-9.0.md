# Installation Version 9.0

1. GitHub Desktop öffnen und sicherstellen, dass keine eigenen ungesicherten Änderungen vorhanden sind.
2. Den gesamten Inhalt dieses Ordners in den lokalen Repository-Ordner `bs-rohrbach-erasmus` kopieren.
3. Vorhandene Dateien ersetzen. Den Ordner `.git` im bestehenden Repository nicht löschen.
4. In GitHub Desktop die Änderungen prüfen.
5. Commit: `Version 9.0 – CMS-gesteuerte Reisekarte`
6. `Push origin` klicken.
7. Cloudflare-Deployment abwarten und danach die Website mit Strg+F5 laden.

Hinweis: Die Karte wird im CMS unter **Karte & Orte** verwaltet.
