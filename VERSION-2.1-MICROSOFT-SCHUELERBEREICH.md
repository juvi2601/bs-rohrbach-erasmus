# Version 2.1 – Microsoft-Schülerbereich

## Aktive Links

- **Fotos hochladen:** Microsoft Forms, Anmeldung mit Schulkonto
- **Schülergalerie:** geschützte SharePoint-Seite, Anmeldung mit Schulkonto

## Datenschutz

Schülerfotos werden nicht auf der öffentlichen Netlify-Seite gespeichert. Die öffentliche Website enthält nur die beiden Microsoft-365-Links. Hochgeladene Bilder sollen vor einer Veröffentlichung von der Reiseleitung geprüft werden.

## Später einen direkten Galerie-Link eintragen

Sobald in SharePoint eine eigene Fotobibliothek oder Galerieseite eingerichtet ist, kann in `config.js` der Wert `studentGalleryUrl` durch deren direkten Link ersetzt werden.
