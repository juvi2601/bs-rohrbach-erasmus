# Version 2.3 – vollständige Bearbeitung über das CMS

Im Redaktionsbereich unter **Reiseportal → Alle Seitentexte & Bilder** können nun sämtliche festen Texte, das Titelbild, Hotelbild, Genussbilder, Links, Schaltflächen, Wetterorte, Navigation und Fußzeile verändert werden.

Weitere Menüpunkte verwalten News, Tagesprogramm samt Bildern, öffentliche Galerie, Karte/Orte samt Bildern, Downloads und FAQ.

## Fotos ändern
Im jeweiligen Feld auf **Bild auswählen** klicken. Neue Medien werden in `images/uploads` gespeichert. Bestehende Originalfotos wurden nicht verändert.

## Veröffentlichen
Nach dem Speichern im CMS auf **Veröffentlichen** klicken. Netlify aktualisiert die Seite danach automatisch.

## Datenschutz
Schülerfotos bleiben ausschließlich im geschützten Microsoft-365-/SharePoint-Bereich.
